LPAI eNPU4 SDK¶

LPAI eNPU4 SDK is released as separate QPM package and to be used to test LPAI offline generated model.

Linux Instructions:¶

The LPAI add-on is currently not present in the base Hexagon SDK. If you would like to install it follow the steps below. Add-ons are accessible through QPM Linux

1. To get the latest version, please Update product-list using qpm-cli –catalog-refresh
2. qpm-cli --license-activate hexagonsdk5.x
3. qpm-cli --install hexagonsdk5.x -v 5.4.0.0 or later
4. qpm-cli --license-activate lpai
5. qpm-cli --install lpai –version 2.4.0.0

Note

Documentation can be found here, after installing the LPAI addon using the below instructions: $HEXAGON_SDK_ROOT/ addons/LPAI/eNPU4/<version>/docs/guide.html

Windows Instructions:¶

The LPAI add-on is currently not present in the base Hexagon SDK. If you would like to install it follow the steps below. Add-ons are accessible through QPM Windows GUI.

1. To get the latest version, please refresh QPM GUI catalog.
2. Activate the license for hexagon sdk 5.x and install 5.4.0.0 or later
3. Activate the license for LPAI under “Qualcomm Hexagon SDK Products”
4. Install the LPAI v 2.4.0.0

Note

Documentation can be found here, after installing the LPAI addon using the below instructions: $HEXAGON_SDK_ROOT/ addons/LPAI/eNPU4/<version>/docs/guide.html
