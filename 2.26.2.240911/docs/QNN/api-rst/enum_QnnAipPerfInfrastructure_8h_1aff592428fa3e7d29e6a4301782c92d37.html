Enum Documentation¶

enum QnnAipPerfInfrastructure_PowerConfigOption_t¶

This enum defines all the possible performance options in Dsp Performance Infrastructure that relate to setting up of power levels.

Values:

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_DCVS_ENABLE = 1¶

config enum implies the usage of dcvsEnableConfig struct. For dcvs v2, if not provided, will set to false

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_SLEEP_LATENCY = 2¶

config enum implies the usage of sleepLatencyConfig struct

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_SLEEP_DISABLE = 3¶

config enum implies the usage of sleepDisableConfig struct

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_DCVS_POWER_MODE = 4¶

config enum implies the usage of dcvsPowerModeConfig struct. If not provided, power save mode will be used

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_DCVS_VOLTAGE_CORNER = 5¶

config enum implies the usage of dcvsVoltageCornerConfig struct

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_BUS_VOLTAGE_CORNER = 6¶

config enum implies the usage of busVoltageCornerConfig struct

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_CORE_VOLTAGE_CORNER = 7¶

config enum implies the usage of coreVoltageCornerConfig struct

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_RPC_CONTROL_LATENCY = 9¶

config enum implies the usage of rpcControlLatencyConfig struct

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_RPC_POLLING_TIME = 10¶

config enum implies the usage of rpcPollingTimeConfig struct this config is only supported on V69 and later if enabled, this config is applied to entire process max allowed is QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIG_MAX_RPC_POLLING_TIME us

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_HMX_TIMEOUT_INTERVAL_US = 11¶

config HMX timeout interval in us. The HMX is turned off after the set interval time if no interaction with it after an inference is finished.

enumerator QNN_AIP_PERF_INFRASTRUCTURE_POWER_CONFIGOPTION_UNKNOWN = 0x7fffffff¶

UNKNOWN config option which must not be used.
